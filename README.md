# remote-control-nodejs

### Running
`npm run start`

### Port
Port is stored in the env file and by default is `8080`
`PORT=8080`

### Commands:
- mouse_up {y px}
- mouse_down {y px}
- mouse_left {x px}
- mouse_right {x px}
- mouse_position
- draw_circle {px}
- draw_rectangle {px} {px}
- draw_square {px}
- prnt_scrn
